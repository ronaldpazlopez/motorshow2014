  • Airbags

    7 airbags que significan el máximo de seguridad para sus ocupantes.

  • Passive Start

    Sistema inteligente que abre las puertas del auto y arranca sin utilizar la llave. El vehículo se pondrá en marcha si la llave se encuentra al interior. (Versión Titanium)

  • Active Park Assist

    Sistema que funciona a base de sensores que miden el espacio hasta encontrar el lugar adecuado. El auto se estaciona solo utilizando estos sensores que miden el espacio. (Versión Titanium)

  • Control ElEctrÓNIco de Estabilidad (ESC)

    Sistema que asegura el control del auto al conductor, utilizando sensores que detectan situaciones de riesgo, para controlarlas mediante el sistema de frenos. En situaciones extremas, disminuye la potencia del motor.

  • Control de Tracción (TCS)

    Los sensores ABS aseguran la tracción del vehículo evitando que giren libremente para que el conductor no pierda el control en ningún momento. En casos de patinar, bloquea la rueda que perdió adherencia devolviéndole el control al conductor.
